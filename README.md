## 1. 서비스 개요

> 명지대학교 멋쟁이사자처럼 13기 아기사자 모집 사이트

2025 명지대학교 멋쟁이사자처럼 13기 아기사자 모집을 위해 개발된 **지원 사이트**입니다.

- 동아리 소개를 위한 **랜딩 페이지**
- 지원을 위한 **지원폼**
